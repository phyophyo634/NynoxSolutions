1. IT Strategic Plan Service

Nynox will provide an IT Strategic Plan services that is setting out the IT future states, a clear roadmap and necessary resources to achieve the organizations with a comprehensive and effective IT strategic plan for the next two to five years:
Scope of Work:

Review and assess the IT current state:

  • Review of current IT systems & infrastructure
  • Review of current IT organization & management

Define and develop IT future state, roadmap & investment plan

  • Define the IT future state and propose key IT projects to be implemented
  • Define projects’ priorities and implementation roadmap
  • Define IT Investment Plan
  • Recommend the execution plan based on IT Strategic Plan

2. IT Governance Service

Nynox will provide IT Governance services that focus on the IT strategy, structure, process, and standards.
Scope of Work:

Review and assess the current IT strategy and performance:

  • Review of IT strategy, structure, and processes
  • Review of IT systems and infrastructure
  • Advise the appropriate IT services and for the business department
  • Advise the appropriate IT methodology to improve the performance

Define and Develop IT governance:

  • Define IT services and decisions
  • Propose IT governance structure, process, and standardization
  • Recommend implementation roadmap
  • Recommend the execution plan based on IT Strategic Plan

3. Data Analytic Service

Nynox will provide Data Analytic services that could realize the situation of the business by transforming from business transaction data to business analytical insight results.
Scope of Work:

Review and assess the current IT strategy and performance:

  • Review of the data storage systems (e.g. Data on-premise, Data on cloud)
  • Review of data processing systems (e.g. Master data management, Data warehousing methodologies, Data Integration system)
  • Review of reporting design and required dashboard standards

Define the standard data architecture and data processing system

  • Define data architecture design, data platform, data standardization
  • Develop data integration system using Microsoft Azure Data Factory (ADF)
  • Develop Data Warehouse and provide BI Services
  • Develop Data Analytic Service using Microsoft Azure Analysis Service (AAS) and Power BI platform
  • Develop Master Data Management Platform


4. Web Application Development Service


Nynox will provide web application development services to local and international business organizations. The applications for social & financial organization, e-commence systems and a part of ERP systems are expected to develop by Nynox’ team.
Scope of Work:

Investigation and Estimation the systems:

  • Review the nature of Business
  • Review the functions and features of the Business
  • Review the business model and system architecture
  • Estimate the time and cost of the system development

Planning for system development:

  • Define the project roadmap for clear vision of development process and steps
  • Define the development plan
  • Define the information architecture for the system
  • Prepare the proposal for the system development

Design, Implementation and Testing the system:

  • Define the UI/UX design for the system
  • Define the system environment (e.g. Cloud computing, On-site development)
  • Define the Database architecture and storage structure
  • Define front-end interface and back-end business logic structure
  • System security checking and testing

Deployment and Maintenance the system

  • Deploy the system architecture platform
  • System installation and configuration
  • Review the user’s feedbacks
  • Improve the systems
  • Maintenance and support the system
